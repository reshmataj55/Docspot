# 🩺 DocSpot – Seamless Appointment Booking for Health

DocSpot is a full-stack healthcare appointment booking application that enables patients to find doctors, book appointments, and manage their healthcare experience through a simple and user-friendly interface.

## 🚀 Features

- User Registration & Login
- Browse Doctors
- Book & Manage Appointments
- Secure Authentication
- Admin Dashboard
- Responsive Design

## 🛠️ Tech Stack

**Frontend**
- React.js
- HTML5
- CSS3
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

## 📂 Project Structure

```
DocSpot/
├── client/

└── README.md
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/BNeeraja123/Docspot-Seamless-Appointment-Booking-For-Health.git
```

Install dependencies:

```bash
cd client
npm install

cd ../server
npm install
```

Run the application:

```bash
# Backend
npm start

# Frontend
cd client
npm start
```
